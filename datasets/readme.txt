Both Yelp and Douban contain three files, which are described as follows:
1. groupid_events.dat
Description:
Each line of this file represents the events that a group has attended
Format:
Group ID\tEvent1,Event2...

2. groupid_users.dat
Description:
Each line of this file represents the members (users) of a group
Format:
Group ID\tUser1,User2...

3. user_events.dat
Description:
Each line of this file represents the events a user has attended
Format:
User ID\tEvent1,Event2...

4. social.dat
Description:
Each line of this file represents the friend relationship of a user
Format:
User ID\tFriend1,Friend2...
